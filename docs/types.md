# Supported Types

## Integer

- `lt: int | None`
- `gt: int | None`
- `ge: int | None`
- `le: int | None`

## String

- `min_length: int | None`
- `max_length: int | None`
- `length: int | None` (Default: `10`)

## Floating Point

- `lt: int | None`
- `gt: int | None`
- `ge: int | None`
- `le: int | None`
- `decimal_places: int | None`

## Boolean

## UUID

**Supported formats**<br/>
**`format`** = `uuid1` | `uuid3` | `uuid4` | `uuid5` | `uuid6` | `uuid7` | `uuid8`
